## 模板引擎的基本使用
 
  视图函数：

```
  # 模板的操作
  def tmp_demo(request):
    # 分配数据
    context = {
        'var':'iloveyou',
        'arr':['aa','bb','cc'],
        'user':{'username':'admin','email':'admin@qq.com'},
        'con':'<h1 style="color:red;">iloveyou</h1>',
        # 'con':'<script>alert("hello")</script>'
        'num':12
        }
    
    # 加载模板
    return render(request,'tmp/1.html',context)
    # return render(request,'tmp/2.html',context)
```

- 模板运算

> 加

{% raw %}
```
  { {  value|add:10  } }
  note:value=5,则结果返回15
```
{% endraw %}

> 减

{% raw %}
```
  { {  value|add:-10  } }
  note:value=5,则结果返回-5，加一个负数就是减法了
```
{% endraw %}

> 乘

{% endraw %}
```
  {%  widthratio 5 1 100 % }
  note:等同于：(5 / 1) * 100 ，结果返回500，
  withratio需要三个参数，它会使用参数1/参数2\*参数3的方式进行运算，进行乘法运算，使「参数2」=1
```
{% endraw %}

> 除

{% raw %}
```
  {%  widthratio 5 100 1 % }
  note:等同于：(5 / 100) * 1,则结果返回0.05,和乘法一样，使「参数3」= 1就是除法了。
```
{% endraw %}

- 自定义标签或过滤器

  1.在应用中创建一个 templatetags 文件夹,定义一个__init__.py(可写可不写)

  2,在 templatetags 包中定义 一个模块文件 diytags.py

  ```
    from django import template
    register = template.Library()

    # 自定义过滤器
    @register.filter
    def yc_upper(val):
        # print ('val from template:',val)
        return val.upper()

    # 自定义标签
    from django.utils.html import format_html
    @register.simple_tag
    def jia(a,b):
        res = int(a) + int(b)
        return res
  ```

  3,在需要使用的模板中 导入 自定义的模块

{% raw %}
```
  <ul>
    {% load diytags%}
    <li>自定义过滤器转大写:{{ var|yc_upper }}</li>
    <li>自定义标签加法运算:{% jia 10 2%}</li>
  </ul>
```
{% endraw %}

- 模板继承

  编写继承的html：

{% raw %}
```
  {% extends 'base.html'%}

  {% block title%}
  <title>2.html</title>
  {% endblock%}

  {% block con%}
      2.html的内容显示
  {% endblock%}
```
{% endraw %}

  被继承的html：

{% raw %}
```
  <!DOCTYPE html>
  <html>
  <head>
      <meta charset="utf-8">
      {% block title%}
      <title>模板文件</title>
      {% endblock%}
  </head>
  <body>
      <div style="height: 130px;background: #369;">
          <a href="/">首页</a>
      </div>
      <div style="height: 400px;">
          {% block con%}
              内容
          {% endblock%}
      </div>

      <div style="height: 130px;background: #ccc;"></div>
  </body>
  </html>
```
{% endraw %}
