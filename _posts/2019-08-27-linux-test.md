## 管理文件和目录的命令

1. pwd 
输出当前目录，即当前所在地址。

2. cd
改变所在目录
```
  cd /   转到根目录。
  cd ～  转到/home/user的用户目录中。
  cd /usr/test  转到根目录的usr的test目录下。
```

3. ls
用来查看目录的内容。
```
  -a  列举目录中的全部文件，包括隐藏文件。
  -l  列举目录中细节，包括权限、所有者、组群、大小、创建日期、文件是否链接。
  -f  列举的文件显示文件类型。
  -r  逆向，从后向前地列举目录中的内容。
  -s  按文件大小排序。
  -R  递归，列举当前目录下的所有子目录内的内容。
  -h  以人类可读的方式显示文件的大小，如k、M、G作单位。
```

4. cat
可以用来合并文件，也可以在屏幕上显示整个文件的内容。
```
  把textfile1的文档内容加上行号后输入 textfile2 这个文档里：
  cat -n textfile1 > textfile2
  把textfile1和textfile2的文档内容加上行号（空白行不加）之后将内容附加到 textfile3 文档里：
  cat -b textfile1 textfile2 >> textfile3
```

5. grep
在文件中查找一个特定的字符串。默认区分大小写。
-i 不区分大小写。

6. touch
用来创建新文件。

7. cp
用来拷贝文件。
```
  cp t.txt documnent/t 
  把文件t.txt复制到document目录下。
  -i  如果文件覆盖目标中的文件，会提示确认。
  -r  会递归复制整个目录树、子目录以及其它。
  -v  显示文件的复制进度。
```

8. mv
用来移动文件。
```
  -i  如果文件覆盖目标中的文件，会提示确认。
  -f  不提示目标中是否有同名文件，直接强制移动。
  -v  显示文件的移动进度。
```

9. rm
用来删除文件。
```
  -i  提示是否删除。
  -f  不提示是否删除，直接删除。
  -v  显示问件的删除进度
  -r  递归删除某个目录以及其中所有文件和子目录。
```

10. rmdir
用来删除空目录。

# wps字体设置
1. 到windows环境下将所需字体拷贝到Linux下（C:/Windows/fonts)
2. 进入拷贝到Linux的目录中（cd 或直接进入右键“打开终端”）
3. 进入目录中，执行(把字体放到“其他位置/计算机/usr/share/fonts”)
  sudo cp * /usr/share/fonts
4. 为字体赋予可执行权限
  chmod 755 *
5. 生成字体的索引信息
  sudo mkfontscale（提示命令未找到：yum install mkfontscale）
  sudo mkfontdir（提示命令未找到：yum install fontconfig）
6. 更新字体缓存
  sudo fc-cache
7. 重启系统reboot
