wif适配器无法在Ubuntu被发现

发表于2019-07-16|更新于2022-06-06|Linux
|字数总计:778|阅读时长:3分钟|阅读量:

​ 安装Ubuntu系统后发现wifi的适配器不能发现，所以要下载相应的驱动并安装。

  • 查找相应的网卡的信息

安装8821CE无线网卡驱动

  • 升级ubuntu内核到4.14

​ $ sudo apt autoremove --purge

 	make
  	sudo make install
 	sudo modprobe -a 8821ce
