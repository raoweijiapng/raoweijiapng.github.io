Linux基础命令测试和wps字体设置

发表于2019-07-27|更新于2022-06-06|Linux
|字数总计:1.3k|阅读时长:4分钟|阅读量:

测试

安装deb

sudo dpkg -i deb文件

管理文件和目录的命令

  1. pwd
    输出当前目录，即当前所在地址。

  2. cd
    改变所在目录

  1. ls
    用来查看目录的内容。
  1. cat
    可以用来合并文件，也可以在屏幕上显示整个文件的内容。

  1. grep
    在文件中查找一个特定的字符串。默认区分大小写。
    -i 不区分大小写。

  2. touch
    用来创建新文件。

  3. cp
    用来拷贝文件。

  1. mv
    用来移动文件。
  1. rm
    用来删除文件。
  1. rmdir
    用来删除空目录。

磁盘命令

  1. mount
    挂载文件系统。
    不加任何参数，显示已挂载的文件系统和目录。

  2. umount
    卸载已挂上的文件系统。

  3. df
    用来检查硬盘分区和已挂载的文件件系统的磁盘空间。

  4. du
    用来显示文件目录或大小。

  5. fsck
    用来检查和修复linux文件系统。

文件备份和压缩命令

  1. bzip2/bunzip2
    扩展名为bz2的压缩/解压缩工具

  2. gzip/gunzip
    扩展名为gz的压缩/解压缩工具

  3. zip/unzip
    扩展名为zip的压缩/解压缩工具

  4. tar
    创建备份和归档。

关机和查看系统信息的命令

  1. shutdown
    正常关机。

  2. reboot
    重启计算机。

  3. ps
    查看在计算机中有哪些程序执行，及其执行的情况。

  4. top
    可以查看目前程序的执行情景和内存使用。

  5. kill
    用来终止一个正在执行中的进程。

  6. date
    用来显示、设定和修改现在的时间和日期。

  7. cal
    显示月历及年历。

管理使用者和设立权限的命令

  1. chmod
    改变许可权限。

  2. useradd
    增加用户。

  3. su
    用来切换用户。

线上查询命令

  1. man
    查询和解释一个命令的使用方法，以及这个命令的说明事项。

  2. locate
    定位文件和目录。

  3. whatis
    用来查询某个命令的含义。

文件阅读的命令

  1. head
    查看文件的开头部分。

  2. tail
    查看文件的结尾。

  3. less
    可以一页一页地查看信息。

  4. more
    可以一页一页地查看信息。

网络操作命令

  1. ftp
    用来传输文件。

  2. bye
    结束连线并结束程序。
    在ftp模式下，输入bye即可中断目前的连线作业，并结束ftp的执行。

  3. ping
    使用icmp传输协议，发出要求回应的信息，若远程主机的网络的没有什么问题，就会回应信息。

  4. telne
    进行远程登录。

  5. rlogin
    进行远程登录。与telne相似。

  6. netstat
    显示linux的网络情况。

定位、查找文件的命令

  1. which
    从path环境变量的目录中找出命令的位置，并显示完整的路径名称。

  2. whereis
    找出命令的位置以及它的可执行文件、源代码文件。

  3. find
    按条件搜索，并执行一定的动作。

  4. locate
    从索引库中查找文件。

  5. updatedb
    更新索引库。

其他命令

  1. echo
    用来在显示器上输出一段文字，或者在文件上加上一段文字。

  2. clear
    清除显示器。

  3. passwd
    用来修改用户的密码。

  4. lpr
    把指定的文件发送到打印队列。

  5. lprm
    取消打印队列中的作业。

wps字体设置

  1. 到windows环境下将所需字体拷贝到Linux下（C:/Windows/fonts)
  2. 进入拷贝到Linux的目录中（cd 或直接进入右键“打开终端”）
  3. 进入目录中，执行(把字体放到“其他位置/计算机/usr/share/fonts”)
    sudo cp * /usr/share/fonts
  4. 为字体赋予可执行权限
    chmod 755 *
  5. 生成字体的索引信息
    sudo mkfontscale（提示命令未找到：yum install mkfontscale）
    sudo mkfontdir（提示命令未找到：yum install fontconfig）
  6. 更新字体缓存
    sudo fc-cache
  7. 重启系统reboot
